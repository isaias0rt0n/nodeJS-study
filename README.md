# **Iniciando com nodeJS** :label:
### Projeto de construção de uma API básica com nodeJS
## Instalação:rocket:
```
#clone o repositório
git clone https://github.com/isaias0rt0n/nodeJS-study

#entre no diretório do projeto
cd backend

#Instale as dependencias do projeto
yarn install

#execute o projeto
yarn dev
```
